# TesloShop Api

1. Clonar el proyecto.

2. Instalar los paquete de package.json

```bash
bun install
```

3. Clonar el archivo `.env.template` y cambiarlo por `.env` y poner valor de nuestra variable de entorno.

4. Levantar en modo de desarrollo:

```bash
node --run dev
````
